14 ## Goals
15 
16 As a goal of this work I want to create complete navigation stack without using global position such as GPS. For local navigation it uses Multi-State Constraint Kalman Filter which is at the time of writing state-of-the-art method. It also has a great computation power to accuracy ratio. Drawback of this approach is that still accumulates (relatively small) drift during time. To compensate for this I want to use mechanism that uses loop closures. It will be based on principles used in ORB-SLAM. I have quite a bit experience with it and it works great.
17 
18 By combining these two approaches I want to create navigation stack that will be able to perform life-long navigation using very cheap hardware and with low energy demands. It should be able to run on battery. It should also be able to run on CPU only.
19 
20 Goal list:
21  - Accurate navigation
22  - Low-cost hardware
23  - Life-long navigation
24  - Low-energy demand (battery)
25  - Global drift compensation (loop closure)

27 ## Datasets
28 
29 For development purpose, I use [MIT Stata Center Data Set](http://projects.csail.mit.edu/stata/index.php). It contains rosbag files recorded from PR2 robot.
30 
31 Each bag file is quite large because it contains laser scans. They are not needed for the purpose of this work, so I created a filtered version of them using command
32 
33 `$ rosbag filter 2011-01-18-06-37-58.bag pr2.bag 'topic in ("/wide_stereo/left/image_rect", "/wide_stereo/left/camera_info", "/torso_lift _imu/data", "/tf", "/robot_pose_ekf/odom_combined")'`
34 
35 By the way. Can you believe how hard it is to find publically available bagfile that is recorded using some cheap hardware? C'mon!

37 ## Installation
38 
39 To install this you need to have installed and working ROS. Then it should be fairly easy to build and run.
40 
41 ```
42 git clone https://github.com/tomas789/tonav.git
43 cd tonav
44 mkdir build
45 cd build
46 cmake ..
47 make
48 ```
49 
50 ## Run
51 
52 I currently don't provide any roslaunch file. Just run `roscore` and then run Tonav
53 
54 ```
55 ./tonav --image <image_topic> --camerainfo <camerainfo_topic> --imu <imu_topic>
56 ```
57 
58 For [MIT Stata Center Data Set](http://projects.csail.mit.edu/stata/index.php) I run it using
59 
60 ```
61 ./tonav --image /wide_stereo/left/image_rect --camerainfo /wide_stereo/left/camera_info --imu /torso_lift_imu/data
62 ```
